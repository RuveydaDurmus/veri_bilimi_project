## Proje Konusu
Bu proje, Online Retail veri seti kullanılarak e-ticaret müşteri davranışlarının analiz edilmesini ve RFM + K-Means yöntemiyle müşteri segmentasyonu yapılmasını amaçlar.

1. Repository'yi bilgisayarına indir.
2. `data/` klasörüne `Online Retail.xlsx` dosyasını koy.
3. Jupyter Notebook veya Google Colab ile notebook dosyasını aç.
4. Hücreleri sırayla çalıştır.
